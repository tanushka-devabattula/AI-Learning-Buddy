# AI Learning Buddy

AI Learning Buddy is an AI-powered educational web application that helps users learn any topic through simple explanations, real-life examples, quizzes, and interactive AI conversations.

## Features
- Explain Concepts
- Real-Life Examples
- Generate Quiz
- Ask Anything

## Technologies Used
- Python
- Streamlit
- OpenRouter API

## Deployment
- Streamlit Community Cloud
